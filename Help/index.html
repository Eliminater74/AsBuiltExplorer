1. Overview

AsBuiltExplorer is your "Swiss Army Knife" for Ford As-Built data. Whether you are adding features to your F-150, fixing a replacement module, or just exploring what your truck can do, this tool simplifies the complex hex data found in FORScan.

2. As-Built Comparison (The Core Feature)

This tab allows you to load multiple vehicle files and see exactly what makes them different. This is how "Mods" are discovered.

How to Use:

  1. Load File 1 (Base): Click Browse to load your own backup file (.ab or .abt).
  2. Load File 2 (Target): Load a file from a vehicle that has the features you want (e.g., a Raptor or Platinum).
  3. Analysis: The tool will instantly align the files by Module Address (e.g., 726-01-01).

Understanding the Colors:

  • White: Identical data. No changes needed.
  • Red: Value Mismatch. The actual configuration data is different. This is where the magic happens.
  • Yellow: Checksum Difference Only. The data is the same, but the checksum byte (last byte) differs. You can usually ignore these.
Pro Tip: Right-click on any line to copy the data or "View Definition" to see if we know what that specific address controls.

3. Calculators & Tools

Stop guessing checksums or manually converting hex.

Checksum Calculator

When you change a digit in FORScan, the last two digits (the Checksum) become invalid. This tool fixes that.

  • Summation (Legacy): Used by older modules (GEM, etc). Simple addition of bytes.
  • CRC-8 / CRC-16 (Modern): Used by newer modules (IPMB, APIM, etc). Select the correct algorithm from the dropdown.

VIN Decoder

Find out exactly what options a donor VIN has.

  • Offline Mode: Instant decoding using our internal database. Secure and fast.
  • Online (NHTSA): Check the box Use Online API to validte data against government records (req internet).
  • Window Sticker: Use the generated link to grab the original PDF window sticker (if available).

4. Vehicle Database (Your Fleet)

Keep track of all the vehicles you work on.

  • Importing: Drag & Drop .ab files into the app, or use the Scan Folder button to bulk-import your entire backup folder.
  • Search: Quickly find verify if you have a "2018 Lariat" file to compare against without digging through folders.
  • Deducer: (Advanced) Select multiple vehicles with a specific feature (e.g., "Has Sunroof") to let the tool find the common bits responsible for it.

5. Mod Database & Warnings

We've built in a library of common mods with instructions.

Critical Warning: Always pay attention to the "Instructions" panel. Some mods require specific hardware!
Example: Enabling "Global Windows" on a standard GEM module will not work and may cause battery drain if the hardware relays are missing.
