AsBuiltExplorer Help

Introduction

Welcome to AsBuiltExplorer, a powerful toolkit designed for "Digital Archaeology" of vehicle configuration data. This tool allows you to compare, analyze, and modify As-Built data files (.ab, .abt, .ucds) to discover hidden features and understand vehicle configurations.

Compare As-Built

This is the core feature for discovering differences between vehicles.

  • Load Files: Load up to 4 different As-Built files to compare side-by-side.
  • Smart Comparison: Differences are automatically highlighted.
    • Red Highlight: Value mismatch.
    • Yellow Highlight: Checksum difference only.
  • Known Features: The application attempts to translate hex data into human-readable feature names (e.g., "Heated Seats") based on the internal database.
  • Right-Click Actions: Right-click any row to Identify Feature, Copy Data, or Edit.

Calculators & Tools

Essential utilities for modifying hex data.

  • Checksum Calculator: Automatically calculate the correct checksum byte for modified data.
  • Hex/Decimal Converter: Quick conversion between numbering systems.
  • Bit Helper: Visualize specific bits within a byte (useful for toggle flags).

Vehicle Database & Deducer

Analyze patterns across multiple vehicles.

Tip: Use the "Scan Folder" button to import your entire collection of As-Built files into the local database for powerful searching.
  • Deducer: Filter vehicles by features (e.g., "Find all F-150s with Sunroof") to isolate the specific address/bit controlling that feature.
  • Analyze Library: Compare a specific file against your entire library to find unique or common settings.
